# qrscanner

QRScanner provides a simple command-line interface to interact with the program. The following commands are available:

scan: This command allows you to scan a QR code from an image file.

openlink: This command opens the URL contained in the last scanned QR code.

save: This command saves the last scanned QR code data to a text file.

browse: This command opens a file dialog to select an image file for scanning.

You can run the program and enter one of the above commands to use the respective functionality.
